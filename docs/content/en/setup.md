Add `@nuxtjs/module-test-utils` devDependency to your project:

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn add --dev @nuxtjs/module-test-utils
  ```

  </code-block>
  <code-block label="NPM">

  ```bash
  npm install --save-dev @nuxtjs/module-test-utils
  ```

  </code-block>
</code-group>
